# 剧本项目技能卡（skill.md）

## 1. 项目定位
- 项目名：深渊回响：反诈迷途
- 类型：现代反诈题材、硬核推理本
- 目标体验：通过时间线还原、动机对冲与物证链闭环，锁定唯一真凶。

## 2. 固定世界观锚点（改稿时默认不可动）
- 场景：宏图集团顶层秘密楼层（会议室、档案区、杂物间、消防通道、走廊）。
- 机制：记忆弧光用于清除近期记忆，是全案核心转折。
- 结构：聚会前铺垫 -> 当天动线 -> 冲突爆发 -> 装置启动 -> 失忆搜证 -> 锁凶复盘。
- 真凶：张磊。
- 主题：反诈警示 + 受害者与加害者边界失守。

## 3. 角色功能分工（推理位）
- 张磊：真凶与掩盖者，负责制造误导与伪证路线。
- 苏晴：核心反派位（非最终真凶），承担高嫌疑与反转压力。
- 陈凯：技术嫁祸位，承担“会操作但不敢启动”的可疑性。
- 王秀兰：组织者与道德灰区，提供动机起点。
- 林晓雅：被操控棋子位，承担信息泄露与恐慌行为。
- 刘敏：护短干扰位，推动情绪冲突。
- 赵强：沉稳探查位，提供装置外部线索，不做鲁莽偷取动作。

## 4. 逻辑审查清单（每次改稿必过）
1. 时间一致性
- 同一事件在所有文档中的时间戳必须一致（含手表与楼层时钟换算）。
- 启动、生效、昏迷、苏醒四个节点必须有唯一版本。

2. 空间一致性
- 谁在何时能否到达何处，必须满足路径与耗时。
- 监控覆盖区、盲区、消防通道可见性要统一口径。

3. 机制一致性
- 记忆弧光的触发条件、生效范围、冷却/时延规则必须唯一定义。
- 一旦定义为全楼层生效，就不能在别处写“仅会议室生效”。

4. 证据一致性
- 每条关键物证必须有唯一来源、唯一落点、唯一发现轮次。
- 嫁祸证据要区分“被害人原本携带”与“真凶后投放”，不可双版本并存。

5. 动机一致性
- 角色行为必须与其既定性格、收益/风险比匹配。
- 如设定赵强为沉稳探查，则避免出现“临场冲动偷零件”版本。

6. 单人本与公本一致性
- 单人本新增机制不可覆盖公本基础规则。
- 若单人本增加新线索，必须在公本中有可验证入口。

## 5. DM引导准则
- 先让玩家建立“共同失忆 + 各自隐瞒”的共识，再放技术线索。
- 先排除“不会/不能”再比较“有动机但无时间”，最后锁“动机+能力+机会”三重重合者。
- 禁止DM临场新增规则，只能解释既有规则。

## 6. 修订优先级（从高到低）
1. 装置规则统一（是否全楼层生效、是否需要特定区域、启动与生效时延）。
2. 监控系统统一（监控室存在性、可见区域、可删除素材边界）。
3. 赵强线索统一（金属碎片来源与“只探查不偷取”的设定一致）。
4. 时间轴统一（尤其14:48-14:53关键窗口）。
5. 单人本措辞统一（避免给出与公本冲突的新硬设定）。

## 7. 后续写作模板（用于新增章节）
每个新章节按以下字段输出：
- 章节目标：本段要解决的推理问题。
- 时间戳：楼层时钟 + 个人手表（如有）。
- 公开行动：所有人可见。
- 隐藏行动：仅特定角色知道。
- 物证变化：新增/转移/销毁。
- 监控可见性：可见、盲区、缺失原因。
- 可被质证点：玩家可追问并验证的句子。

## 8. 版本管理建议
- 采用单一“设定母本”（建议以情节总稿为准）作为唯一真相源。
- 其他文档（人物档案、单人本）只做派生，不独立改机制。
- 每次改动后，至少执行一次“时间线对账表 + 物证对账表”检查。
