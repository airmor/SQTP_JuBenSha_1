# 《深渊回响：反诈迷途》第二版真实游玩流程

> 依据第二版参考文献整理，默认以第二版设定为准。该流程面向主持人使用，重点记录真实游玩节奏、搜证轮次、阶段性推理与线索发放顺序。

## 一、开场统一口径

### 0.1 主持人开场说明
- 先明确本局基调：七人均有黑历史，玩家要通过搜证还原“谁在什么时间做了什么”。
- 先统一时间口径：顶层时钟比玩家手表快 1 分钟；监控系统时间另有独立偏差，后续由线索卡说明。
- 只提示一个核心事实：本局的关键窗口都围绕“会议室、档案室、杂物间、消防通道、电梯口”展开。

### 0.2 发放内容
- 角色本。
- 第一张公共线索卡：监控时间差与删除日志。

### 0.3 主持人目标
- 让玩家先接受“时间差”和“监控缺口”这两个推理前提。
- 不要过早解释记忆弧光的完整原理，只允许玩家先感受到“有装置、但有人在掩盖”。

## 二、第一阶段：失忆苏醒与身份确认

### 1.1 建议时长
- 10到15分钟。

### 1.2 场景目标
- 玩家完成角色认领、基础关系梳理和初始动机互相试探。
- DM 只引导他们说出“我记得什么、我不记得什么、我最担心什么”。

### 1.3 阶段重点
- 这时不进入深搜证，只做公开讨论。
- 让玩家先建立三个认知：
  - 有人想销毁证据。
  - 有人知道楼层结构和监控。
  - 有人和杂物间装置有关。

## 三、第一轮搜证：会议室与走廊

### 2.1 建议时长
- 15分钟。

### 2.2 发放线索卡
- 会议室散落证据包。
- 张磊受贿笔记碎片。
- 赵强脚边金属碎片。

### 2.3 发放节奏
- 先发共享卡，再发偏个人视角的卡。
- 若桌子讨论过热，先让他们看共享卡，再用个人卡修正判断。

### 2.4 搜证重点
- 会议室为什么会在冲突前后出现散乱文件。
- 谁更像“知道太多的人”。
- 谁在走廊和会议室之间频繁进出。

### 2.5 第一阶段推理目标
- 初步锁定“会议室内的公开冲突”只是表象。
- 把嫌疑范围从“所有人都像凶手”收缩到“少数几个人真正接触过关键路径”。
- 让玩家开始怀疑：真正的关键不是谁最激动，而是谁能离开会议室再回来。

## 四、第二阶段：档案室搜证与人设反转

### 3.1 建议时长
- 15分钟。

### 3.2 发放线索卡
- 王秀兰心理研究报告缺页。
- 林晓雅倒卖信息纸条。
- 苏晴签字交接单。
- 档案室垃圾桶话术灰烬。

### 3.3 搜证重点
- 档案室不是单纯藏资料的地方，而是每个人身份崩塌的地方。
- 重点让玩家看到：
  - 王秀兰不是普通受害者，她知道很多内部心理资料。
  - 林晓雅有私下挪用、倒卖行为。
  - 苏晴对内部文件流转非常熟。
  - 这些人都在档案室里留下过自己的痕迹。

### 3.4 第二阶段推理目标
- 把“苏晴是不是普通前台”彻底推翻。
- 让玩家意识到“受害者组织”里没有纯白角色。
- 让王秀兰、苏晴、林晓雅这条线先互相咬起来，但先不要直接定真凶。

## 五、第三阶段：杂物间与消防通道

### 4.1 建议时长
- 15分钟。

### 4.2 发放线索卡
- 记忆弧光装置说明。
- 装置按钮隐形粉末与陈凯指纹。
- 刘敏装置照片。
- 消防通道推门痕迹与鞋印。

### 4.3 搜证重点
- 这一轮是整局的定凶轮次。
- 让玩家围绕四个问题找答案：
  - 谁知道装置是什么。
  - 谁真的碰过装置。
  - 谁具备离场再回场的行动条件。
  - 谁最有动机销毁自己的罪证。

### 4.4 第三阶段推理目标
- 从“谁接触过装置”推到“谁能完成启动动作”。
- 把陈凯从“最像凶手”拆成“最像被嫁祸的人”。
- 最终把张磊锁成唯一闭环人物。

## 六、终局推理与结案

### 5.1 建议时长
- 20分钟。

### 5.2 终局逻辑链
- 时间差统一。
- 监控缺口成立。
- 会议室侧门与消防通道的动线成立。
- 装置启动、延迟生效、蓝光消失、全员失忆成立。
- 张磊具备动机、路径、能力、掩盖手段四项闭环条件。

### 5.3 主持人收束方式
- 先让玩家口头锁凶，再由主持人补完全时间线。
- 最后再揭示陈哲“意识后门”的科学解释，完成故事闭环。

## 七、推荐总节奏

- 开场与身份确认：10到15分钟。
- 第一轮搜证与推理：15到20分钟。
- 第二轮搜证与推理：15到20分钟。
- 第三轮搜证与定凶：15到20分钟。
- 终局复盘与结尾：20分钟左右。

## 八、主持人可直接照做的发卡顺序

1. 开场先发：监控时间差与删除日志。
2. 第一轮发：会议室散落证据包、张磊受贿笔记碎片、赵强脚边金属碎片。
3. 第二轮发：王秀兰心理研究报告缺页、林晓雅倒卖信息纸条、苏晴签字交接单、档案室垃圾桶话术灰烬。
4. 第三轮发：记忆弧光装置说明、装置按钮隐形粉末与陈凯指纹、刘敏装置照片、消防通道推门痕迹与鞋印。
